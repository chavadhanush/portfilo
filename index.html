TODO-APP-SPRING-BOOT

The To-Do Application is a simple yet powerful tool developed using Spring Boot, designed to help users manage their tasks efficiently. This project demonstrates the fundamental concepts of a Spring Boot application, including RESTful APIs, data persistence, and dependency injection. It leverages Spring Data JPA for seamless database interactions and uses an H2 in-memory database for ease of development and testing. The application follows a layered architecture, dividing responsibilities among the controller, service, and repository layers, ensuring clean and maintainable code. Users can perform CRUD (Create, Read, Update, Delete) operations.
